# 🎬 Movie Recommendation System

This project is a simple movie recommendation system built using Machine Learning techniques. It recommends movies based on genre similarity.

## 🚀 Features
- Recommends similar movies based on input
- Uses content-based filtering
- Simple and fast execution

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn

## ⚙️ How it Works
- Movie genres are converted into numerical form using CountVectorizer
- Cosine similarity is used to find similar movies
- Based on user input, similar movies are recommended

## ▶️ How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run the program:
   python main.py

3. Enter a movie name when prompted

## 📌 Example
Input: Inception  
Output: Interstellar, Avengers, Joker
